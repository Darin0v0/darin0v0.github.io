System_Plik

Projekt systemu zarządzania plikami

O projekcie

System_Plik to aplikacja do zarządzania plikami z interfejsem graficznym, napisana w Pythonie przy użyciu biblioteki Tkinter.

Przeglądaj pliki

Intuicyjna nawigacja po strukturze katalogów

Wyszukuj pliki

Szybkie wyszukiwanie plików po nazwie

Operacje na plikach

Kopiuj, przenieś, usuń i zmieniaj nazwy plików

Zrzuty ekranu

Instalacja i uruchomienie

# Klonowanie repozytorium
git clone https://github.com/Darin0v0/System_Plik.git
cd System_Plik

# Uruchomienie aplikacji
python main.py
